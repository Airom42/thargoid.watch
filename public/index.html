Non-Human Signal Sources

Non-Human Signal Sources (NHSS) are found in the following regions of space. Any system in these regions will spawn NHSS near other bodies within the system. These can be found using the Full Spectrum Scanner or by scanning the Navigation Beacon.

Pleiades Nebula

~175ly sphere centered on Merope

Witch Head Nebula

~75ly sphere centered on Witch Head Sector IR-W C1-9

Coalsack Nebula

~60ly sphere centered on Musca Dark Region PJ-P B6-1

California Nebula

~100ly sphere centered on HIP 18390
